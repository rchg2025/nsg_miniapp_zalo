# 🎨 TẠO APP ICON TỪ LOGO

## 📐 **Yêu cầu Zalo Mini App:**
- **Kích thước:** 512x512 pixels
- **Format:** PNG với nền trong suốt hoặc trắng
- **Chất lượng:** High resolution (300 DPI)
- **Style:** Flat design, dễ nhận biết ở kích thước nhỏ

## 🔧 **Cách resize logo:**

### **Phương pháp 1: Sử dụng online tools**
1. Truy cập: https://www.iloveimg.com/resize-image
2. Upload logo của trường
3. Chọn "By pixels" 
4. Nhập: Width = 512, Height = 512
5. Giữ "Keep aspect ratio" để không bị méo
6. Download file PNG

### **Phương pháp 2: Sử dụng Paint (Windows)**
1. Mở logo bằng Paint
2. Resize → Pixels
3. Nhập 512 x 512
4. Save as PNG

### **Phương pháp 3: Photoshop/GIMP**
1. Mở logo
2. Image → Image Size
3. Width: 512px, Height: 512px
4. Export as PNG

## 🎯 **Tối ưu logo cho app icon:**

Logo trường hiện tại rất phù hợp vì:
✅ Thiết kế tròn (phù hợp với icon)
✅ Màu sắc nổi bật (xanh navy + đỏ)
✅ Text rõ ràng ở kích thước nhỏ
✅ Thể hiện danh tính trường học

**Không cần thay đổi gì, chỉ cần resize!**
