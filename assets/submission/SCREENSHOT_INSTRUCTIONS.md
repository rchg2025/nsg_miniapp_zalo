## 🔧 **HƯỚNG DẪN KỸ THUẬT:**

### **A. Cài đặt trình duyệt để chụp:**
```
1. Mở Chrome/Edge
2. Truy cập: http://localhost:3000
3. Nhấn F12 (Developer Tools)
4. Nhấn Ctrl+Shift+M (Device Mode)
5. Chọn iPhone 12 Pro (390x844)
6. Zoom = 100%
```

### **B. Cách chụp screenshot:**
```
1. Chuẩn bị màn hình
2. Nhấn Ctrl+Shift+P
3. Gõ "screenshot" 
4. Chọn "Capture full size screenshot"
5. Hoặc dùng "Capture screenshot" cho viewport hiện tại
```

### **C. Yêu cầu chất lượng:**
- **Format:** PNG hoặc JPG
- **Kích thước:** Tối thiểu 390x844px
- **Chất lượng:** HD, rõ nét
- **Nội dung:** Không chứa thông tin cá nhân thật

### **D. Đặt tên file:**
```
1_homepage.png
2_news_page.png  
3_admission.png
4_major_detail.png
5_profile.png
6_admin_dashboard.png (optional)
```
