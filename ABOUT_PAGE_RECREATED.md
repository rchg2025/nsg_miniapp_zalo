## 🔧 **TECHNICAL DETAILS:**

### **Components sử dụng:**
- `Page` - Container chính
- `Header` - Thanh tiêu đề với back button
- `Box` - Layout containers
- `Text`, `Text.Title` - Typography
- `Icon` - Icon system
- `Button` - Action buttons

### **Navigation:**
- Back button: `navigate(-1)`
- Internal routing: `navigate('/settings')`, `navigate('/profile')`
- External link: `window.open('https://namsaigon.edu.vn', '_blank')`

### **Styling:**
- Tailwind CSS classes
- ZMP UI design tokens
- Consistent spacing và colors
- Shadow effects cho depth
